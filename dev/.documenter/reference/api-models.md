
# Coral Models API {#Coral-Models-API}

`PolyGrowthModel` and `PolySurvivalModel` are the concrete model types returned by [`load_models`](/reference/api-interface#Kora.load_models) and accepted by [`initialize_reef`](/reference/api-reef-state#Kora.initialize_reef). They are opaque containers. Interact with them through the I/O and simulation functions rather than constructing them directly.
